Pre-Install Preperations

Before we get started installing Linux onto our computer, there are some things we need to take care of. I will explain these things in this blog post. In the next post, I will walk you through the process of actually installing it.

We will be installing Ubuntu Linux, which is a ‘distribution’ of Linux that is suitable for beginners. In case you don't know what a distribution is, it's simply a convenient package of Linux along with some commonly used software. There are several different distributions (or ‘distros’ as they are commonly called) that include software for different uses. For now, we will use Ubuntu.
