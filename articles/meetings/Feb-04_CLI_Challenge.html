Feb 4th: CLI Challenge

Today we practiced our CLI skills using this website to solve some fun problems using bash commands.

Try to do the first 10 or so problems.

While solving the problems, first try to recollect the commands you learned (and their options!) and think of creative ways to apply that knowledge. If you are still stuck, use the man command to check the manual page of the relavant command. If, however, you are still stuck, refer to our earlier blog posts for ideas. Finally, if you still can't solve it, after having given it your best, check the solutions in the dropdown menu above the problem on that website. There will be many solutions that solve the problem. All of them work, but usually, the shorter and simpler the solution, the better it is.

If you have any questions, like “Why does this solution work?”, send me an email (or just google it :) ).
